排序算法

排序算法
数据结构与算法
发布日期:   2023-09-17

1 前言

排序算法分为 内部排序 和 外部排序，其中:

内部排序：是数据记录在内存中进行排序；

外部排序：因为要排序的数据很大，内存一次性不能容纳全部的排序记录，在排序中需要访问外存。

ps:简单排序：冒泡排序、选择排序、插入排序； 高级排序：希尔排序、归并排序、快速排序。

2 常见（内部）排序算法

常见的内部排序算法有：冒泡排序、选择排序、插入排序、希尔排序、归并排序、快速排序、堆排序、基数排序等。

2.1 冒泡排序

2.1.1 算法步骤

（1）比较相邻的元素。如果第一个比第二个大，就交换他们两个；

（2）对每一对相邻元素做同样的工作，执行完毕后，找到第一个最大的值；

（3）重复以上步骤，每次比较次数-1，直到不需要比较。

2.1.2 代码实现

#include <iostream>
using namespace std;

template<typename T>
void bubble_sort(T arr[], int len)
{
    // 排序的总趟数=元素个数-1；
    for(int i = 0; i < len - 1; i++)
    {
        // 每趟对比次数=元素个数-当前排序趟数-1；
        for(int j = 0; j < len - i -1; j++)
        {
            // 相邻元素比较
            if(arr[j] > arr[j + 1])
            {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}

void printArr(T arr[], int len)
{
    for(int i = 0; i < len; i++)
    {
        cout << arr[i] << " ";
    }
    cout << endl;
}
int main()
{
    int arr[] = {61, 17, 29, 22, 34, 60, 72, 21, 50, 1, 62};
    int len = sizeof(arr) / sizeof(arr[0]);
    bubble_sort(arr, len);
    printArr(arr, len);

    float arrf[] = { 17.5, 19.1, 0.6, 1.9, 10.5, 12.4, 3.8, 19.7, 1.5, 25.4, 28.6, 4.4, 23.8, 5.4 };
    int len = sizeof(arr) / sizeof(arr[0]);
    bubble_sort(arr, len);
    printArr(arr, len);

    return 0;
}

2.1.3 性能分析

时间复杂度：O(N^2)

空间复杂度：O(1)

2.2 选择排序

2.2.1 算法步骤

（1）在每一次遍历的过程中，都假定第一个索引处的元素是最小值，并和其他索引处的值进行比较，如果当前索引处的值大于其他索引处的值，则假定其他索引处的值为最小值，最后可以找到最小值所在的索引；

（2）交换第一个索引处和最小值所在的索引处的值。

2.2.2 代码实现

#include <iostream>
#include <algorithm>
using namespace std;

template<typename T>
void selection_sort(vector<T>& arr)
{
    // 排序的总趟数=元素个数-1；
    for(int i = 0; i < arr.size() ; i++)
    {
        // 定义一个变量，记录最小元素所在的索引，默认为参与选择排序的第一个元素所在的位置
        int minIndex = i;
        for(int j = i + 1; j < arr.size(); j++)
        {
            // 比较 最小索引处minIndex处的值 与 j索引处的值 的大小
            if(arr[minIndex] > arr[j])
            {
                minIndex = j;
            }
        }

        //交换最小元素所在索引minIndex处的值和第一个位置处的值
        swap(arr[i], arr[minIndex]);
    }
}
int main()
{
    vector<int> intv;
    intv.push_back(5);
    intv.push_back(1);
    intv.push_back(4);
    intv.push_back(2);
    intv.push_back(3);

    selection_sort(intv);
    for(auto v : intv)
    {
        cout << v << " ";
    }
    cout << endl;
    return 0;
}

2.2.3 性能分析

时间复杂度：O(N^2)

空间复杂度：O(1)

2.3 插入排序

2.3.1 算法步骤/原理

（1）把所有的元素分为两组，已经排序的和未排序的；

（2）找到未排序的组中的第一个元素，向已经排序的组中进行插入；

（3）倒叙遍历已经排序的元素，依次和待插入的元素进行比较，直到找到一个元素小于等于待插元素，那么就把待插入元素放到这个位置，其他的元素向后移动一位。



ps:假设排序前的数组的第一个的元素为已排序，其余为未排序

2.3.2 代码实现

#include <iostream>
using namespace std;

template<typename T>
void insertion_sort(T arr[], int len)
{
    // 索引0处为已排序，待排序的元素索引从1开始
    for(int i = 1; i < len; i++)
    {
        for(int j = i; j >= 0; j--)
        {
            // 比较索引j处的值和索引j-1处的值，
            // 如果j-1处的值大于j处的值，则交换位置；
            if(arr[j-1] > arr[j])
            {
                swap(arr[j-1], arr[j]);
            }
            // 否则找到合适的位置退出循环
            else
            {
                break;
            }
        }
    }
}

void printArr(T arr[], int len)
{
    for(int i = 0; i < len; i++)
    {
        cout << arr[i] << " ";
    }
    cout << endl;
}

int main()
{
    int arr[] = {61, 17, 29, 22, 34, 60, 72, 21, 50, 1, 62};
    int len = sizeof(arr) / sizeof(arr[0]);
    printArr(arr, len);
    return 0;
}

2.3.3 性能分析

时间复杂度：O(N^2)

空间复杂度：O(1)，稳定的排序算法

ps:元素集合越接近有序，插入排序算法的时间效率越高

2.4 希尔排序

希尔排序是插入排序的一种，又称“缩小增量排序”，是插入排序算法的一种更高效的改进版。

2.4.1 算法步骤/原理

（1）选定一个增长量h，按照增长量h作为数据分组的依据，对数据进行分组；

（2）对分好组的每一组数据完成插入排序；

（3）减小增长量，最小减为1，重复第二步操作。



增量h的确定规则：

// h初始化
int h = 1;
while(h < 数组的长度/2)
{
    h = 2h + 1;
}

// 减小规则
h = h / 2;

2.4.2 代码实现/核心代码

void shell_sort(T arr[], int len)
{
    // 1.根据数组长度确定增长量h的初始值
    int h = 1;
    while(h < len/2)
    {
        h = 2*h + 1;
    }
    // 2.希尔排序
    while(h >= 1)
    {
        // 排序
        // 2.1 找到待插入的元素
        for(int i = h; i < len; i++)
        {
            // 2.2 将待插入的元素插入到有序数列中
            for(int j = i; j >= h; j-=h)
            {
                // 待插入的元素是a[j],比较a[j]和a[j-h]
                if(a[j-h] > a[j])
                {
                    swap(a[j-h], a[j]);
                }
                else
                {
                    break;
                }
            }
        }
        
        // 减小h的值
        h = h / 2;
    }
}

2.4.3 性能分析

时间复杂度O(N^1.5) [不好计算]

空间复杂度O(1)

ps: 非稳定排序算法,当h>1时都是预排序，目的是让数组更接近于有序。当h==1时，数组接近有序的，排序就会更快。

2.5 归并排序

2.5.0 递归

  • 定义：在方法内部调用方法本身；
  • 作用：可以把一个大型复杂问题层层转换为一个与原问题相似的、规模较小的问题来求解。递归策略只需要少量的程序就可以描述出解题过程所需要的多次重复计算，大大减少程序的代码量。
  • 注意事项：递归必须要有边界条件能够让递归结束，因为每一次调用都会在栈内存开辟新的空间，重复执行方法，如果递归的层级太深，很容易造成栈内存溢出。
  • 代码示例(n的阶乘)：
long factorial(int n)
{
    if(n == 1)
    {
        return 1;
    }
    return factorial(n-1);
}

2.5.1 算法步骤/原理

(1) 尽可能的一组数据拆分成两个元素相等的子组，并对每一个子组继续拆分，直到拆分后的每个子组的元素个数是1为止；

(2) 将相邻的两个子组进行合并成一个有序的大组；

(3) 不断地重复步骤2，直到最终只有一个组为止。
